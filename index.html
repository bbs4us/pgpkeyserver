OpenPGP Public Keyserver {{ site.baseurl }}

This server is a host and relay for pgp public keys via the sks-keyserver pool. You may connect to this network by adding pool.sks-keyservers.net to the keyserver option in your pgp software.

The sks-keyserver pool is a pool of approximately one hundred (100) servers that store, relay and publish pgp keys uploaded to each server. When you upload your key to one sks-keyserver pool server, it will quickly be relayed to the other sks-keyserver servers active in the pool.

Extract a key from the pool

You can find a key by typing in some words that appear in the userid (name, email, etc.) of the key you're looking for, or by typing in the keyid in hex format ("0x…").

About this service

This web page is provided as a convenient alternative to the usual methods of interacting with the keyserver. Key submission and retrieval are usually handled by key management software (such as GnuPG) on your local system.

This keyserver supports both the traditional email interface pks (for key submission only), and the more common hkp interface, for interaction with PGP key management software.
